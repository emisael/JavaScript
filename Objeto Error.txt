Objeto Error

Exibe informações sobre o erro ocorrido. É composto de duas propriedades e um método
Propriedades: message e name;
Método: toString().

A propriedade message mostra os detalhes do erro.
A propriedade name mostra o tipo de erro
O método toString mostra o tipo de erro e a mensagem separados por dois pontos.

Tipos de erros
-EvalError: Indica um erro utilizando o método eval(). O metodo eval permite executar uma linha javascript no formato de string;
-RangError: Indica que o script encontrou um número que está fora do intervalo usando na expressão corrente;
-ReferenceError: Erro de referencia, geralmente uma referência a uma variável ou valor não definido ou não encontrado;
-SintaxeErro: Ocorre quando se escreve uma instrução javascript errada;
-TypeErro: Indica a utilização de operando diferente do operando esperado;
-URIError: Indica o uso errado de funções URI
