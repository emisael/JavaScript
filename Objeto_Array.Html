Objetos Array

São um tipo especial de variáveis utilizadas para armazenar diversos valores ao mesmo tempo. É possível amarenar matrizes de dados.
Cada elemento de um array é chamado de indice, sendo eles numerados inciando a partir de 0.

Declaração de um array.
var nomes = new Array();

Atribuindo valores a um índice do Array;
nomes[0] = "Emisael";

var nomes = new Array();
nomes[0] = "Mateus";
nomes[1] = "Marcos";
nomes[2] = "Lucas";
nomes[3] = "João";
nomes[4] = "Atos dos Apóstolos";
