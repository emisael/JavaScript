Expressões Regulares


Expressões regulares descrevem padrões de caracteres. São utilizadas para buscar padrões ou sequências que podem ser utilizados em substituições por exemplo.
Uma expressão regular é definida da seguinte forma:
/exp/g(x). Onde (x) determina o modo de busca
Para (x) = i : Busca ignorando case sensitive, ou seja, considera tanto maiúsculas como minúsculas.

*g: Retorna todos os padrões encontrados na string em questão.

var nom = Emisael Cavalcante da Silva;

Verificando se uma expressão regular está dentro de uma string com o método match; var rgexp1 = /Ca/gi;

nom.match(rgexp1) =

Verificando (de uma outra forma) se uma expressão regular está dentro de uma string com o método test,
que retorna true caso a expressão regular seja encotrata na string ou false caso não seja encotrada.
var nom2 = new RegExp('xa','i');
nom2.test('Emisael Cavalcante da Silva') =

que retorna a parte localizada caso ela seja encontrada
var nom2 = new RegExp('va','i');
