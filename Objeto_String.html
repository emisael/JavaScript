lenght: retorna a quantidade de caracteres de uma string;
Ex: nome.lenght =

indexOf(): retorna o índice (posição) da primeira ocorrência de um caractere ou sequência de
caractere dentro de uma string;
Ex: nome.indexOf('a') =
*Nao obtendo resultado o método retornará -1;

lastIndexOf(): retorna o índice (posição) da útima ocorrência de um caractere ou sequência de
caractere dentro de uma string;
Ex: nome.indexOf('a') =
*Nao obtendo resultado o método retornará -1;

search(): é igual ao método indexOf(), retorna o índice (posição) da primeira ocorrência de um caractere ou sequência de
caractere dentro de uma string;
Ex: nome.indexOf('e') =
*Nao obtendo resultado o método retornará -1;

slice(): retorna parte de uma string. A parte de string que se deseja obter é indicada pelos
dois parâmetros de slice(posicao_inicial,posicao_final);
Ex: nome.slice(5,13) =
Se for omitido o segundo parâmetro, a string será mostrada a partir da posição indicada pelo primeiro parâmetro.
Ex: nome.slice(5) =
Se for colocado apenas um parâmetro e ele for negativo, será exibido a quantidade de caracteres indicada pelo parâmetro
contando do final para o início. Ex: nome.slice(-5) =

substr(): retorna parte de uma string. Com a parte que se deseja obter indecada pelos dois parâmetros de substr(posicao_inicial,qtd_caractere).
O substr() não aceita indices negativos.
Ex: nome.substr(3,2) =

substring(): Semelhante ao método slice() com a diferença de não aceitar índice negativo
Ex: nome.substring(0,7) =

replace(): substitue um valor específico por outro valor em um string
Ex: nome.replace('_','#') =
Por padrão, replace() substritue apenas a primeira ocorrência, para substituir todas as ocorrencias dentro da string, use /termo_procurado/g.
Ex: nome.replace(/-/g,'#') =

toUpperCase(): converte uma string para maiúscula.
Ex: nome.toUpperCase() =

toLowerCase(): converte uma string para minúscula.
Ex: nome.toLowerCase() =

concat(): junta duas ou mais cadeias de caracteres.
Ex: nome.concat(' programador',' JavaScript', ' e ', 'Delphi') =
*concat() pode ser usado em vez do sinal de (+).

charAt(): retorna um caractere localizado em um índice específico de uma string.
Ex: nome.charAt(5) =

charCodeAt(): retorna o Unicode de um caractere específico de uma string.
Ex: nome.charCodeAt(5) =

split(): converte uma string para um array. Na string a ser convertida para um array, deve haver um caractere que
funcionará com separador. Tal caractere deve se repedir várias vezes, pois ele é quem determinará as várias partes de array.
Ex: nome.split('_') =
Após convertido em um array, pode-se acessar todos os elementos do mesmo normalmente.
Ex
var arr = nome.split('_');
var i;
for (i = 0; i < arr.length; i++){
document.getElementById('view14e').innerHTML += arr[i];
}
*Resultado

*Se o separador for omitido, split() retornará um array com um único elemento contendo toda a string, este elemento estará na posição [0] do array.
Ex: nome.split() =
*Se o separador for indicado com vazio (''), o array retornado será um array de caracteres individuais.
