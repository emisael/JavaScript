Operador de comparação igual (==), verifica se um valor é igual a outro.
x = 5; y = 5 => x == y:

Operador de comparação extatamente igual (===), verifica se um valor é igual a outro
e se também é do mesmo tipo.
x = 5; y = "5" => x === y:

Operador de comparação diferente (!=), verifica se um valor é diferente de outro.
x = 5; y = 5 => x != y:

Operador de comparação exatamente diferente (!==), verifica se um valor é diferente de outro
e se também é de outro tipo.
x = 5; y = "5" => x !== y:

Operador de comparação menor que (<), verifica se um valor é menor que outro.
x = 5; y = 3 => x < y:

Operador de comparação maior que (>), verifica se um valor é maior que outro.
x = 5; y = 3 => x > y:

Operador de comparação menor ou igual a (<=), verifica se um valor é menor ou igual que outro.
x = 5; y = 6 => x <= y:

Operador de comparação maior ou igual a (>=), verifica se um valor é maior ou igual que outro.
x = 5; y = 5 => x >= y:
