O operador AND (&&), veirifica se todas as situações são verdadeiras, caso sejam, retorna true
x = 5; y = 7; z = 3 w = 1 => x < y && z > w :

O operador (||), veirifica se ao menos uma das situações é verdadeis, caso seja, retorna true
x = 9; y = 6; z = 4 w = 4 => x == y || z == w :

O operador NOT (!), verifica se uma situação é falsa, caso seja retorna true
x = 8; y = 8 => !(x==y):
