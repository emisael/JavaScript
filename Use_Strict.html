Use Strict

Recurso que determina que o JavaScript seja executado em modo mais rigoroso.
Usando este recurso será gerado um erro de (exception) quando:
Caso alguma parte do código seja escrito de forma errada;
Seja usada uma variável que não foi declarada...

var num1 = 4,
num2 = 5,
num3 = 6,
num4 = 10;
soma = num1 + num2 + num3 + num4;
document.getElementById('view1').innerHTML = soma;


'use strict'
var num1 = 4,
num2 = 5,
num3 = 6,
num4 = 10;
soma = num1 + num2 + num3 + num4;
document.getElementById('view1').innerHTML = soma;
